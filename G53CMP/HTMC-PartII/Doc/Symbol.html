Symbol
Description

Symbols: information about named entities. Symbols are used to keep track of information about defined and declared named entities, like variables, procedures, functions, and types. They are used in the symbol table and MTIR.

There are two kinds of symbols: term-level symbols, denoting entities like variables and procedures, and type-level symbols, denoting types.
