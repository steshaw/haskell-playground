ParseMonad
Description
Monad for scanning and parsing. The scanner and parser are both monadic, following the design outlined in the Happy documentation on monadic parsers. The parse monad P is built on top of the diagnostics monad D, additionally keeping track of the input and current source code position, and exploiting that the source code position is readily available to avoid having to pass the position as an explicit argument.
